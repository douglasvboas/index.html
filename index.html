O que é HTML?

HTML é uma linguagem utilizada na criaçao de paginas da web.

O que é necessário para se criar uma pagina em HTML?

É necessário um navegador como o google chrome e um editor de texto como o bloco de notas.

O que é uma Tag?

Tags sao instruçoes que modificam a estrutura da página.

O que é um Link?

Link é uma estrutura utilizada para ligar documentos em HTML.
